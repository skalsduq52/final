프로젝트 계획서 : 
1차 회의를 통해 그룹웨어, 쇼핑몰, SCM등 주제 선정을 위한 회의를 진행하고
2차 회의에서 그룹웨어로 주제를 선정하고 메일,결재,일정,예약,게시판 등
각 기능정의 및 분업을 진행했고 3차회의로 깃서버 안정화, 진행상황을 공유하며
프로젝트를 진행했습니다.

분석/설계 : 그룹웨어의 특성에 맞게 임직원 테이블을 먼저 생성하고 그에 따른 다른 기능들인 사원관리/이메일/전자결재/게시판/예약/일정관리 테이블을 생성하여 각 테이블에서의 필요 정보들을 인지하고 분석/설계하였습니다.

[발표자료 작성]
 - 회의록, 계획서부터 시연자료까지 흐름의 순서대로 발표자료 작성
 - 전체 구성은 팀소개, 기획의도, 개발과정, 개발결과를 대분류로 구성
 - 개발과정은 회의록, UML, 유스케이스 다이어그램, ERD자료를 첨부하여 시각화
 - 개발결과는 주요기능별 해당 소스코드와 뷰페이지를 가시화하여 작성
 - 구체적인 기능 작동은 실시간 시연을 통해 발표

구현/테스트 :
1. 사원 관리(로그인&로그아웃, ID&PW 찾기)
- 로그인 & 로그아웃 테스트 완료
- ID & PW 찾기 테스트 완료(PW 찾기 시 EMAIL로 임시
비밀번호 전송)
 
2. 이메일 구현
- 메일 쓰기 및 수정, 삭제 테스트 완료
- 메일 다중 선택 후 읽음, 안읽음, 휴지통으로 상태 변화 테스트 완료
- 메일 작성 시 참조자 검색 테스트 완료

3. 일정 관리 구현
- 월간, 주간, 일간별 일정 관리 테스트 완료
- 당일 일정, 기간 일정 정보 조회 테스트 완료
- 작성자 외 수정 및 삭제 불가능 테스트 완료

4. 공지사항, 건의사항 게시판 구현
- 게시판 등록 및 수정, 삭제 테스트 완료
- 게시물 클릭시 조회소 테스트 완료
- 게시물 페이징, 검색어 테스트 완료

5. 업무 보고 페이지 구현
- 월간 주간별 업무 보고 페이지 구현
- 업무 보고 등록 및 수정, 삭제 테스트 완료
- 업무 보고서 페이징, 검색어 테스트 완료

5. 전자결재 시스템 구현
- 전자결재 등록시 중간 결재자, 최종 결재자 검색 및 드래그드롭으로 추가
테스트 완료
- 결재가 완료 되기 전 대기 문서 목록에 위치 테스트 완료
- 결재 수신 문서, 대기 문서 조회 테스트 완료 

6. 예약 관리 구현
- 회의실, 회사 차량 예약 등록 및 취소, 반납 테스트 완료
- 기간 지정 예약 등록 테스트 완료
- 예약 목록 조회 테스트 완료

7. 조직도 구현
- 사원 별 정보 및 이미지 삽입 테스트 완료
- 사원의 email 클릭시 바로 email 작성 페이지로 이동 테스트 완료
